how it works

the idea

drp is a lightweight drop point for text and files. No account, no login. A key is your address — whoever has it can access the content.

dropping something

From the home page, paste text or drop a file. Pick a key or leave it blank for an auto-generated one. Hit save & get link — you'll land on drp.yourdomain.com/your-key with a shareable URL ready to copy.

accessing a drop

Go to /your-key. If it's text, you can read, copy, or download it. If it's a file, you can download it. You can also edit or replace the content directly on that page.

keys

The key is the only thing protecting your content — think of it as a password and URL in one. Make it long and random for sensitive content, short and memorable for easy sharing. You can rename a key at any time; the old URL stops working immediately.

expiry

Text drops expire 24 hours after creation. File drops expire after 90 days of inactivity. When a drop expires, everything is permanently deleted — the content, the file, and the key is freed up.

your dashboard

The home page shows drops you've created in this browser. This list lives in your browser's local storage — it's never sent to our server. You can export it as a JSON file and re-import it on any device to restore your dashboard.

sync client

The sync client watches a local folder and automatically uploads new or changed files to a key of your choice. Install it with:

pip install watchdog requests
python sync/client.py --setup

It will ask for your host, key, and local folder. From then on, any file you drop in that folder syncs to your drp key automatically.

privacy

drp is open source. Files are stored on Cloudinary. Text is stored in a database. There is no encryption — don't store secrets. The key is security through obscurity only.
